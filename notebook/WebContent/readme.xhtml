Welcome to the Research Notebook Manager. To begin, you should ensure that your web browser is configured as follows:

  • Javascript must be enabled.
  • If you are using Internet Explorer, it is recommended that you use the latest version (IE9).
  • If, with any other browser, you encounter issues or seemingly missing functionality, it is strongly recommended that you upgrade to the latest available stable release version. See below for links to the latest downloads for most popular web browsers.
Using the application:

This application is built to be fully accessible for users restricted to a mouse-only method of input. In order to maximize the efficiency of mouse-based text input, we've created an intuitive on-screen keyboard to assist you. To enter text into any field in this application, simply right-click the field, and the keyboard will appear. Right-click again, and the keyboard will disappear. Go ahead and right-click now on the field below. Try to enter the word "cat".



Good job! The autocomplete functionality built into the keyboard should help to further reduce the effort required to input text via the mouse. Additionally, if you ever discover that the keyboard is obscuring your view of the field you are editing, you are free to click and drag the keyboard to move it to a more desirable location while you enter your text.
